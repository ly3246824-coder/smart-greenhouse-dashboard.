# Smart Greenhouse Monitoring & Control Dashboard 🌿📊

An interactive, data-driven web application designed to monitor and control smart agricultural greenhouse environments in real-time. This project bridges the gap between software development and automated agriculture.

## 🚀 Features
* **Real-time Sensor Telemetry:** Visualizes critical environmental data including Internal Temperature (°C) and Soil Moisture (%).
* **3D Data Visualization:** Implements advanced `Plotly` 3D scatter plots to map greenhouse conditions dynamically.
* **Actuator Status Tracking:** Displays real-time status of critical equipment:
  * **Cooling Fan:** Automated status monitoring for greenhouse temperature regulation.
  * **Drip Irrigation Pump:** Automated status monitoring for precise water management.
* **Responsive UI:** Built with a clean, dark-themed interface using `Streamlit`.

## 🛠️ Tech Stack & Libraries
* **Language:** Python
* **Frontend/Dashboard:** Streamlit
* **Data Visualization:** Plotly (Scatter3d)
